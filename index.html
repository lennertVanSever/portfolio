Approvals - Borealis Digital Studio

Approvals makes it easier to approve or reject items as a manager within Borealis. The purpose is that it becomes so easy that managers approve themself and not delegate it to their assistences.

For this project I worked as a front-end developer to assure the User Interface, experience and integration was coming together. All the components were custom build, no UI library. It was a lot of fun to get the micro transitions just right. One of the challenges was integrating with SAP and having a better mobile experience through adding Progressive Web App (PWA) capabilities. We did the implementation with ReactJS, it was the first time I used React Hooks on a large scale. Approvals is my latest project and it's still under development.
